Privacy & Guide

Privacy

이 브라우저에만 저장됩니다

Profile, tier, 숙박 수, 포인트, 여행 조건은 localStorage에 저장됩니다. GitHub, 호텔, 외부 서버로 보내지 않습니다.

Using My Status

입력하면 화면이 함께 바뀝니다

현재/다음 티어 혜택, Next Tier Path, Elite Ladder, What I’ll Get에 바로 반영됩니다.

도시, 숙박일, 목적을 바꾸면 어떤 프로그램이 유리한지 다시 계산합니다.

Before Booking

예약 전 공식 링크를 확인하세요

조식, 라운지, 업그레이드, late checkout은 호텔·지역·객실 상황에 따라 달라질 수 있습니다.

공개 공유용 기본값은 빈 상태입니다. 이번 버전부터 이전 브라우저 저장값은 자동 초기화됩니다.

Traveler Feedback

Feedback

Privacy-aware

프로필 정보는 자동 전송되지 않습니다

이름, 티어, 포인트, 숙박 수, 여행 계획은 이 브라우저에 남아 있습니다. 피드백에는 사용자가 직접 쓴 내용만 담기도록 설계합니다.

  • 등급/포인트 자동 첨부 없음
  • 브라우저 localStorage와 분리
  • 외부 폼 연결 전 사전 고지
Google Form

피드백은 새 탭에서 작성합니다

앱 안의 Profile, tier, 포인트, 숙박 수는 자동으로 옮겨지지 않습니다. 폼에 직접 적은 내용만 제출됩니다.

Open form
외부 Google Form을 앱 안에 직접 넣지 않고 새 탭에서 엽니다. 현재 멤버십 정보와 여행 조건이 폼에 자동 첨부되지 않도록 하기 위해서입니다.
